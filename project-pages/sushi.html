Interactive Sushi Menu Website
A sushi website that allows you to in real time create and order any type of sushi related food or drink
An interactive menu using Vue and Typescript
Myself and my partner set out to create a sushi menu that any user could create and add custom drink/food orders in real time as well as maintain an account with preferences and previous orders.
Using Vue for a front end user ui, it makes the ui simple and easy to understand. Using typescript we created a robust backend that can handle data retention as well as authentication and validation of users based on log in information. Our website could keep a history of all the menu items you have created as well as recommend menu items that are similar to your likes.
Sushi Gallery
Create your own custom drink/food order at the touch of a button
Account creation for custom orders or to be able to save a list of favorites you have created. Will retain the orders you have previously created and give potential food items similar to those created
Validation for each person for the log in process, allowing for seemless tracking of registered users.
